# 🎙️ Lumira Voice Assistant

## 📌 Overview
Lumira is a Python-based AI voice assistant with a graphical user interface (GUI).  
It can perform tasks like searching information, playing music, fetching news, and more using voice and text commands.

---

## 🚀 Features

- 🎤 Voice command recognition
- ⌨️ Manual text input (chat-style)
- 📰 News updates (live RSS feeds)
- 🌦 Weather information
- 🎬 Movie details (TMDB API)
- 📚 Wikipedia search
- 🎵 Play songs on YouTube
- 🌐 Open websites
- 🔴 Voice interrupt ("stop")
- 💬 Typing animation (AI-style responses)
- ⚡ Shutdown assistant via voice
- 🎯 Status indicator + mic animation

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- SpeechRecognition
- gTTS (Text-to-Speech)
- Pygame (Audio playback)
- Wikipedia API
- TMDB API
- Feedparser (News)
